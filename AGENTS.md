## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Design Context

- **Register:** brand (展示性页面)
- **Platform:** web (响应式)
- **Audience:** Sydney Shibari 社群成员，英文受众
- **Purpose:** 解释竹上金文篆刻的含义
- **Brand Personality:** 克制 · 温润 · 有雅
- **Design Principles:** 文字为主; 克制优雅; 语境贴合社群; 沉稳自信
- **Bg:** `#1a1a1a` dark, accent from inscription gold tones
- **Stack:** Astro 7 + Tailwind CSS v4
